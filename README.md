# Sportify-Music-Analysis using Power-BI & Power-Query
# 🎧 Spotify Music Analysis Dashboard

## 📊 Project Overview
This project presents an interactive music analytics dashboard built using Power BI to analyze data from Spotify. The dashboard explores music trends, artist performance, and track characteristics by transforming raw data into meaningful insights.

Data was cleaned and transformed using Power Query, and multiple analytical dashboards were developed to visualize patterns in music streaming, popularity, and audio features.

---

# 🛠 Tools & Technologies
- 📊 Power BI
- 🔄 Power Query
- 📁 Excel / CSV Dataset
- 📈 Data Visualization
- 📉 Data Analysis

# 📂 Dataset Information
The dataset includes detailed attributes of Spotify tracks such as:

- 🎵 Track Name

- 🔊 Loudness
- 🎹 Key

# 📑 Dashboard Structure

The dashboard is divided into four analytical sections for better exploration of Spotify data.

---

# 📖 1. Introduction
This section introduces the Spotify music analytics project and explains the purpose of the analysis.  

The music streaming industry has transformed how users discover and consume music. With Spotify generating vast amounts of data daily, analyzing this information helps uncover patterns in listening behavior, artist popularity, and music characteristics.

The objective of this dashboard is to leverage Power BI to visualize Spotify data and generate insights that help understand music trends and user preferences.
